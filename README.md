<h2>
💻 Setup
</h2>
</div>

1. Clone the repository:
```bash
git clone https://github.com/Angel-ISO/Crime-io.git

```

2. Install the dependencies:
```bash
pip install -r requirements.txt
```

3. Run the app:
```bash
streamlit run src/Home.py
```

and enjoy! 🚀
